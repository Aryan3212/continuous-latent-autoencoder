# Reference Implementations

This directory holds small, high-value reference material: single-file reference implementations and reimplementation notes that are cheap to keep in-tree and useful when porting ideas into this project. The full vendored upstream repositories (`vjepa2`, `RAE-main`, `le-wm`) were moved out of this repo to `/Users/aryanrahman/Projects/research/reference-implementations-archive` (a sibling directory of the repo) to keep the repo small. Nothing was deleted — the archived trees are intact and can be re-cloned anywhere from upstream.

## Files kept in-tree

- `lejepa_minimal_impl.py` — minimal LeJEPA reference implementation
- `lejepa_training_ready.py` — training-ready LeJEPA reference implementation
- `mhc_reference_implementation.py` — MHC reference implementation
- `zipformer2_implementation.py` — Zipformer2 reference implementation
- `continuous-audio-language-model-encoder.md` — encoder reference notes
- `vjepa2-REIMPLEMENTATION_NOTES.md` — reimplementation notes preserved from the archived `vjepa2` repo
- `le-wm-REIMPLEMENTATION_NOTES.md` — reimplementation notes preserved from the archived `le-wm` repo
